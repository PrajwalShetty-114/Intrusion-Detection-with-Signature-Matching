How to Use the Simulation

Navigate to the Simulation page.
Observe the "Known Signatures" card to see what the IDS is looking for (e.g., virus).
In the "Simulated Network Traffic" input box, type any text.
To see a detection: Type a sentence that includes one of the signatures (e.g., "this is a virus packet").
To see normal traffic: Type a sentence without any signatures (e.g., "this is a normal email").
Click the "Analyze Traffic" button.
Observe the "System Status" box to see if the traffic was marked as "All Clear" or "Intrusion Detected."
